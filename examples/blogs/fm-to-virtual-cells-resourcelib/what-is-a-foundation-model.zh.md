## 一句话定义

一个**基础模型（FM）**，是一个在大而通用的底料上预训练过的神经网络，之后可以被适配到许许多多的下游任务上——靠微调、靠提示，或者当成一个冻结的编码器来用。「基础」这两个字的意思是：预训练才是那笔扛重量的投入，下游的一切都继承自它。

放到生物学里，这就意味着：在数百万细胞 / 数十亿碱基对 / 数十万张切片上训一次；然后拿这套预训练好的表示，去做细胞类型分类、扰动预测、变异效应预测、蛋白结合子设计，或者病理切片解读。

## 五大家族——一眼看全

| 家族 | 分词的对象 | 锚点模型 | 底料规模 | 2026 现状 |
|---|---|---|---|---|
| **单细胞** | 基因 + 细胞作 token（或细胞作有序的基因排名） | scGPT、Geneformer、UCE、scFoundation、CellPLM、TranscriptFormer | 33M–112M 细胞 | 处于危机——清算之后重估扰动预测 |
| **病理** | H&E（或 H&E + IHC）tile → 嵌入 → 切片级聚合 | UNI、Virchow / Virchow2、CHIEF、Prov-GigaPath、PathChat、TITAN | 200M–2B tile，100k–350k 切片 | 最健康——在 ViT-H/14 附近见顶；FDA 路径已开 |
| **基因组** | DNA 碱基或 k-mer | Nucleotide Transformer、DNABERT-2、HyenaDNA、AlphaGenome、Evo2 | 8B–9T token | 分裂——轨迹预测（AlphaGenome）对生成式（Evo2） |
| **蛋白质** | 氨基酸序列 + 结构 token | ESM-2、ESM-3、AlphaFold 3、Proteina Complexa | 27.8 亿蛋白，771B token（ESM-3） | 渐成熟——Proteina Complexa 在 PDGFR 上拿到 63.5% 湿实验命中率 |
| **多模态 / 视觉-语言** | 图像 + 文本共同分词；医学影像或生物学问答 | BioMedCLIP、PathChat-2、MedAgentGym、Med-Gemini | >100 万指令对 | 离临床落地最近——PathChat DX 拿下 FDA 突破性认定 |

## 为什么是这五个？

每个家族，都是由**分词的对象**来定义的。分词怎么选，决定了模型能注意到什么，而那又决定了它能解什么任务。

- **单细胞**：token 是基因（scGPT、scFoundation），或基因的排名（Geneformer），或细胞作序列。模型在转录组上做注意力。
- **病理**：token 是图像 patch（224×224 的 H&E tile）。模型在一张切片上做注意力。
- **基因组**：token 是核苷酸（Evo2）或 k-mer（DNABERT）。模型在 DNA 序列上做注意力。
- **蛋白质**：token 是氨基酸，外加可选的结构（ESM-3 的 7 条 token 轨）。模型在一个蛋白上做注意力。
- **多模态**：token 是图像 + 文本共同编码。模型跨模态做注意力。

**什么*不算*一个基础家族**：专用诊断模型（比如 DeepVariant）、传统 sklearn 流水线、为单一任务做的经典深度学习模型。「基础」这个判据是*在通用底料上做规模化预训练，然后能迁移到许多任务*。

## 单细胞家族哪儿不一样？

这场演讲就活在单细胞家族里，2025 年那场清算也发生在这里。有三个结构性特征把它和其他家族区分开来：

1. **底料更稀疏。**一个细胞有 ~2,000–10,000 个被检测到的基因（人类约 20,000 个蛋白编码基因里）。那是 >50% 的稀疏度。病理 tile 是稠密的像素阵列；DNA 是稠密序列；蛋白也是稠密序列。单细胞表达是货真价实地稀疏——而对这种稀疏建模，并不简单。
2. **下游任务是因果的，不是感知的。**病理 FM 预测的是「这是什么组织？」（感知）。单细胞 FM 被要求去预测「我要是扰动基因 X 会发生什么？」（因果反事实）。2025 年那场清算，本质上就是发现了第二个任务比第一个难得多。
3. **训练语料是偏的。**CELLxGENE Census 里约 70% 来自英语圈的测序中心。Tahoe-100M 是 50 个永生化细胞系。跨供体、跨细胞类型的泛化几乎没被量过，因为去量它的那块底料压根没有大规模存在。

## 「基础模型」这个框架为什么要紧？

三个原因：

1. **算力摊销。**预训练花 $20k–$5M；下游用一次几百美元。当那笔重活是 Arc Institute / DeepMind / EvolutionaryScale 一次性付掉的成本时，学术生物学的经济账就变了。
2. **对大多数临床任务，迁移学习是唯一的路。**罕见癌症、罕见细胞类型、罕见供体——没有一个有足够的带标签数据从头训一个深模型。冻结嵌入 + 小分类器（路线 1）之所以是小实验室的取胜套路，*正是因为*它继承了预训练。
3. **许可这一层要紧。**开放权重（Apache-2.0 / MIT）解锁商用；CC-BY-NC + HF 门禁则限制商用。**这道分叉是按家族来的**：sc-FM 一律宽松；病理 FM 一律受限；基因组 FM 两派分裂。许可要盯紧——UNI 在 2025 年 1 月转成 HF 门禁，就是那个前车之鉴。

## 几个常见误解

**「基础模型 = transformer。」**当下大多数生物 FM 确实基于 transformer，但这个类别并不要求如此。状态空间模型（HyenaDNA、Caduceus）和图增强 transformer，只要满足「预训练一次、适配多处」这个判据，照样算。2026 年的 xVERSE 论文就明确主张走*转录组学原生*（非 transformer）的架构。

**「模型越大 = FM 越好。」**Geneformer V2-104M_CLcancer 用三分之一的算力就追平了 316M 的通用领域模型。xVERSE 比 LM 派生的 sc-FM 高 17.9%。**在生物学里，领域整理和架构选择常常压过参数量**——这和 NLP 不一样，那边往往是规模取胜。

**「FM 一律优于经典方法。」**截至 2026 年 5 月，这话在单细胞里是错的。线性基线（`mean-of-training-perturbations`、`latent-additive + scGPT-embeddings`）在扰动预测上赢过每一个已发表的 sc-FM。见 [为什么线性基线会赢？](why-linear-baselines-win.md)。

**「FM 和智能体 AI 是互相竞争的范式。」**它们是互补的。智能体*用* FM（PathChat-DX = 一个 LLM 智能体把病理 FM 当工具来调）；智能体*训* FM（VCHarness——自主的虚拟细胞构建器）；智能体*在* FM 之上推理（rBio——在 TranscriptFormer 上后训练出来当验证器的 Qwen）。见 [智能体 AI 如何与基础模型相遇](agentic-meets-foundation.md)。
