> *这是[可达性纲领](cell-state-reachability-as-viability-theory.md)的闭环姊妹篇，也接着讲那套在环里抛候选的[扩散 / 流匹配骨干](diffusion-and-flow-matching-for-virtual-cells.md)。如果说[智能体遇见基础模型](agentic-meets-foundation.md)画的是纯软件的「智能体 × FM」模式、[闭环 101 巡礼](closed-loop-virtual-cells-101.md)走的是这个领域的历史，那这篇就补上第三层——那台实体机器——并追问：当闭环的一个角落变成机器人时，它会是什么样子。优先级靠[惊讶 / 不确定性信号](surprise-and-uncertainty-in-cell-fms.md)来跑；项目层面的结论，留在[药物响应智能体闭环 v2](agent-loop-for-drug-response-v2.md)里。*

## 为什么是三层，而不是两层

一个 LLM 智能体是一套策略：它决定下一步做什么，但它没有一个值得信任的世界模型，也没有手。一个基础模型是一个世界模型：它怀着一个对细胞状态习得的密度，但它没有真值——它见过的永远只是已实现的数据——而且它也没有手。一个自驱动实验室手和真值都有：它能把实验跑起来，返回真实答案，可它不知道*哪个*实验值得跑。每一层的长处，恰好是另一层的盲点——这就是为什么真正有意思的系统是把三者都组进来，而不是凑其中两个。智能体是策略，基础模型是代理模拟器，机器是环境兼预言机。这么一框，这套架构其实并不新——它就是基于模型的强化学习，或者等价地说，是一套带习得代理模型、且把真实预言机放进环里的主动学习。新的地方在于：三个角落第一次同时以可部署系统的形态存在了，而大家都低估的恰恰是第三个——那台把预测变成事实的实体机器。

## 经济逻辑：拿便宜的错去套利昂贵的对

整个闭环靠的就是一个不对称。基础模型给你一个*便宜*但*多半错*的答案——一次前向传播只要毫秒，返回一个密度、一个似然、一个预测出来的扰动响应。机器给你一个*昂贵*但*对*的答案——一次真实的 perturb-seq 跑动或一次重编程尝试，要花掉数美元、数周，返回真值。智能体的全部活计就是套利这道落差：尽情花那个便宜的代理模型，去定夺哪个昂贵实验值得做。这就是经典的贝叶斯实验设计——挑那个能让单位成本上的期望信息增益最大的实验（[Foster et al., ICML 2021](https://arxiv.org/abs/2103.02438)）——以及它的近亲贝叶斯优化（[Frazier, 2018 教程](https://arxiv.org/abs/1807.02811)）和主动学习（[Settles, 2009 综述](https://burrsettles.com/pub/settles.activelearning.pdf)）。采集函数才是 FM 真正赚回身价的地方：它抛出候选，而且关键是，给每个候选配一个*不确定性*（那个[惊讶 / 不确定性信号](surprise-and-uncertainty-in-cell-fms.md)），这样智能体就能瞄准那些「真测一下能让模型学到最多」的点，而不是那些模型本来就有把握的点。当候选不仅要高价值、还得*多样*时——你可不想要一批几乎一模一样的实验—— GFlowNets 这类生成式采样器（[Bengio et al., 2021](https://arxiv.org/abs/2106.04399)）就用一个组合拳取代了单挑「最优」的采集。每一轮真实实验的结果都会更新 FM（或者更省钱地，更新一个挂在冻结 FM 上的适配器），密度就在它要紧的地方变锐利，闭环再转一圈。

## 那些已经把环闭上的机器

实体层并不是空想；它只是在化学和材料里走得比细胞生物学远。最早的 **Robot Scientist Adam**（[King et al., *Science* 2009](https://www.science.org/doi/10.1126/science.1165620)）自主对酵母基因功能形成假设、设计实验、在自家硬件上跑、再修订——这是第一个内循环里没有人的端到端闭环。**移动机器人化学家**（[Burger et al., *Nature* 2020](https://www.nature.com/articles/s41586-020-2442-2)）八天里跑了 688 次实验来优化一种光催化剂，靠的是一条自由移动的机械臂，而不是固定的液体处理工作站。**A-Lab**（[Szymanski et al., *Nature* 2023](https://www.nature.com/articles/s41586-023-06734-w)）自主合成了新型无机材料，由 ML 引导，依据是 Materials Project 的凸包。**Coscientist**（[Boiko et al., *Nature* 2023](https://www.nature.com/articles/s41586-023-06792-0)）让一个 GPT-4 智能体掌管一座云实验室，从一个自然语言目标出发去规划并执行真实反应——迄今最干净的「智能体 + 机器」范例——而 **ChemCrow**（[M. Bran et al., *Nat Mach Intell* 2024](https://www.nature.com/articles/s42256-024-00832-8)）则把一个 LLM 接到了一个能驱动合成的工具箱上。这套更大的套路如今有名字了，叫自驱动实验室（[Abolhasani & Kumacheva, *Nat Synth* 2023](https://www.nature.com/articles/s44160-022-00231-0)）。生物这边，手是有的——可编程云实验室（Emerald、Strateos）、生物铸造厂（Ginkgo）、规模化的自动表型组学（Recursion 的 Cell Painting 流水线，[Bray et al., *Nat Protocols* 2016](https://www.nature.com/articles/nprot.2016.105)）——但单细胞读出上的*自主*内循环，还很早。

## 基础模型在这个闭环里到底坐在哪

对虚拟细胞而言，FM 就是那套实验设计闭环里的代理模型，而[可达性纲领](cell-state-reachability-as-viability-theory.md)恰好把每一块接在哪儿都摆清楚了。生成式骨干——一个[扩散或流匹配模型](diffusion-and-flow-matching-for-virtual-cells.md)——抛出候选的可工程化状态或重编程鸡尾酒：一条从成纤维细胞分布通往 iPSC 分布的薛定谔桥路径，是一次候选干预，而不只是一次插值。FM 给每个候选的可达性及其不确定性打分；智能体按期望信息增益给它们排序；自驱动实验室——机器人转染加 scRNA-seq，或者一次 perturb-seq 筛选——去*尝试*排在前面的那一批。结果做了一件计算机版本做不到的事：它用经验刻出「可工程化与禁区」的边界，把一个被断言的层，变成了一个被测量出来的层。这套在细胞上行得通的存在性证明已经发表——那项「把环闭上」的 sc-FM 工作，用迭代采集到的扰动来教 Geneformer，在 T 细胞激活上报出了 PPV 三倍的提升（[101 巡礼](closed-loop-virtual-cells-101.md)里走了一遍）——而 **VCHarness**（[Cheng et al., bioRxiv 2026](https://www.biorxiv.org/content/10.64898/2026.04.11.717183v1)）则用一套多智能体系统把「造模型」那个角落包了起来。一条贯穿始终的线：FM 不是答案，它是那台便宜的模拟器，负责告诉那台昂贵的机器该往哪儿看。

## 诚实的失效模式

有四条注意事项，让这件事不至于被吹过头，而且它们会叠加。第一，**sim-to-real 鸿沟**—— FM 的密度不是实验室的现实，闭环每转一圈都是一次小的分布漂移，所以一个在图谱数据上本来校准良好的代理模型，会随着智能体把实验往图谱从没覆盖过的区域里赶而漂走。第二，**代理模型被钻空子**——给一个智能体配上一个针对有缺陷 FM 的采集函数，它就会找到并利用 FM 的错误，专门提那些「看起来信息量最大」的实验，而它们看起来信息量最大，恰恰是因为模型在那里自信地错了；这和 rBio 推理模型为「FM 当验证器」标出的奖励黑客风险是同一个，只是搬到了「FM 当模拟器」上。第三，**吞吐天花板**——瓶颈是机器，不是模型，所以这个闭环只有在*批量*主动学习下才划算（每轮多个多样的实验），而且得对「一笔预算到底能买到多少个真实循环」抱有狠到家的诚实。第四，**机器速度下的多重比较**——一个自主闭环能在人类还没察觉它在 p-hacking 之前，就生成上百个看着挺像样的命中，这和 CellVoyager 那篇批评为自主分析提出的隐患是同一个，只不过现在多挂了一条机械臂。而压在这四条之下的，是那个显而易见的：自主生物需要生物防护，也需要一道人类闸门，来管住这个闭环被允许在物理上造出什么。

## 这件事会走向哪里

终点是那个闭环可达性实验室。基础模型提出哪些细胞状态应当可工程化；自驱动实验室去尝试它们；那些失败和成功，把「已实现 / 可工程化 / 禁区」的边界重画成一个被测量的对象，而不是一句主张。这才是最终让活力分层*在现实世界里可证伪*、而不只是在计算机里可证伪的东西——你不必去争论某个状态是不是禁区，你看着机器一轮又一轮地造不出它，直到那条边界变成经验性的。没有世界模型的智能体是个瞎子规划者；没有预言机的基础模型是个自信的瞎猜者；没有策略的机器是一场昂贵的随机游走。把它们拧进一个闭环，你就得到了这个领域自第一篇虚拟细胞视角文章以来一直在绕着打转的那个东西：一个会提出生物学、去检验它、再从犯错中学习的系统。引擎已经造好了。剩下要做的，就是在细胞上把这个环闭上——就像化学早已在分子上把它闭上那样。
