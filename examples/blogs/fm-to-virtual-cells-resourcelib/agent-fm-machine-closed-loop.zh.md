## 为什么是三层，不是两层

一个 LLM 智能体是一个策略：它决定下一步做什么，但它没有值得信任的世界模型，也没有手。一个基础模型是一个世界模型：它带着一个习得的、覆盖细胞状态的密度，但它没有真值——它只见过已实现的数据——而且它同样没有手。一个自驱动实验室两者兼有：它能跑实验、返回真实答案，但它毫无头绪 *哪个* 实验值得跑。每一层的长处恰好是另一层的盲点，这正是为什么那些有趣的系统组合了全部三层、而非任意两层。智能体是策略，基础模型是代理模拟器，机器是环境兼预言机。这样框定之后，这套架构并不新——它就是基于模型的强化学习，或等价地说，是带习得代理模型和环内真实预言机的主动学习。新的是：人类历史上第一次，三个角同时作为可部署系统存在，而所有人都低估的那个角是第三个：把一次预测变成一桩事实的实体机器。

## 经济学逻辑：用廉价-错误去套利昂贵-正确

整个闭环跑在一个单一的不对称之上。基础模型给你一个 *廉价的* 答案，它 *很可能错*——一次前向传播花费毫秒级，返回一个密度、一个似然、一个预测的扰动响应。机器给你一个 *昂贵的* 答案，它 *是对的*——一次真实的 perturb-seq 跑批或一次重编程尝试，花费以美元和周计，返回真值。智能体的全部工作就是套利那道差距：自由地花用廉价的代理模型，去决定那些昂贵实验里哪一个值得做。这就是经典的贝叶斯实验设计——挑选那个让每单位成本的期望信息增益最大化的实验（[Foster et al., ICML 2021](https://arxiv.org/abs/2103.02438)）——及其表亲贝叶斯优化（[Frazier, 2018 教程](https://arxiv.org/abs/1807.02811)）和主动学习（[Settles, 2009 综述](https://burrsettles.com/pub/settles.activelearning.pdf)）。采集函数（acquisition function）是 FM 挣得身价之处：它提出候选者，并且关键地，给每个候选者一个 *不确定性*（即 [惊讶 / 不确定性信号](surprise-and-uncertainty-in-cell-fms.md)），这样智能体就能瞄准那些一次真实测量能教会模型最多的点，而不是那些它已经笃定的点。当候选者必须既高价值又 *多样* 时——你不想要一批近乎雷同的实验——像 GFlowNets（[Bengio et al., 2021](https://arxiv.org/abs/2106.04399)）这样的生成式采样器就用一个投资组合替代了单个最优采集。每一轮真实实验的结果都会更新 FM（或者更便宜地，更新冻结 FM 上的一个适配器），密度在它最要紧之处变得更锐利，闭环再转一圈。

## 已经把闭环闭合的那些机器

物理层不是空想；它只是在化学和材料里比在细胞生物学里走得更远。最早的 **Robot Scientist Adam**（[King et al., *Science* 2009](https://www.science.org/doi/10.1126/science.1165620)）自主形成关于酵母基因功能的假设、设计实验、在自己的硬件上运行并修正——这是第一个内层循环里没有人类的端到端闭环。**移动机器人化学家**（[Burger et al., *Nature* 2020](https://www.nature.com/articles/s41586-020-2442-2)）在八天里跑了 688 个实验来优化一种光催化剂，用的是一只自由游走的机械臂而非固定的液体处理仪。**A-Lab**（[Szymanski et al., *Nature* 2023](https://www.nature.com/articles/s41586-023-06734-w)）自主合成了新型无机材料，由 Materials Project 凸包做 ML 引导。**Coscientist**（[Boiko et al., *Nature* 2023](https://www.nature.com/articles/s41586-023-06792-0)）让一个 GPT-4 智能体掌管一个云实验室，从一个自然语言目标出发规划并执行真实反应——迄今最干净的智能体加机器范例——而 **ChemCrow**（[M. Bran et al., *Nat Mach Intell* 2024](https://www.nature.com/articles/s42256-024-00832-8)）把一个 LLM 接上了一个能驱动合成的工具箱。这种更宽的模式现在有了名字，自驱动实验室（[Abolhasani & Kumacheva, *Nat Synth* 2023](https://www.nature.com/articles/s44160-022-00231-0)）。在生物学这一侧，手是存在的——可编程云实验室（Emerald, Strateos）、生物铸造厂（Ginkgo），以及规模化的自动化表型组学（Recursion 的 Cell Painting 流水线，[Bray et al., *Nat Protocols* 2016](https://www.nature.com/articles/nprot.2016.105)）——但作用于单细胞读出的 *自主* 内层循环仍处于早期。

## 基础模型在闭环里究竟坐在哪里

对虚拟细胞而言，FM 是那个实验设计循环里的代理模型，而 [可达性纲领](cell-state-reachability-as-viability-theory.md) 精确地展示了每块拼图插在哪里。生成式骨干——一个 [扩散或流匹配模型](diffusion-and-flow-matching-for-virtual-cells.md)——提出候选的可工程化状态或重编程鸡尾酒：一条从成纤维细胞分布到 iPSC 分布的 Schrödinger 桥路径，是一次候选干预，而不仅仅是一次插值。FM 给每个候选者打出它的可达性及其不确定性；智能体按期望信息增益给它们排序；自驱动实验室——机器人转染加 scRNA-seq，或一次 perturb-seq 筛选——*尝试* 排在最前的那一批。结果做了一件计算机版本做不到的事：它们用经验刻出可工程化对禁区的边界，把一个被断言的层级变成一个被测量的层级。这在细胞上行得通的存在性证明已经发表——那项"closing the loop"的 sc-FM 工作，从迭代采集的扰动里教会 Geneformer，并在 T 细胞激活上报告了三倍的 PPV 提升（在 [101 巡礼](closed-loop-virtual-cells-101.md) 里走读过）——而 **VCHarness**（[Cheng et al., bioRxiv 2026](https://www.biorxiv.org/content/10.64898/2026.04.11.717183v1)）把那个建模的角包进了一个多智能体系统。贯穿全篇的主线：FM 不是答案，它是那个告诉昂贵机器该往哪里看的廉价模拟器。

## 诚实的失效模式

四条告诫让这一切不至于被吹过头，而且它们会叠加。第一，**sim-to-real 鸿沟**——FM 的密度不是实验室的现实，闭环的每一轮都是一次小的分布偏移，于是一个在图谱数据上校准良好的代理模型，会随着智能体把实验引向图谱从未覆盖的区域而漂移。第二，**代理模型被利用**——给一个智能体一个建立在有缺陷的 FM 之上的采集函数，它就会找到并利用 FM 的错误，提出那些看上去信息量最大的实验，恰恰因为模型在那里笃定地错了；这与 rBio 推理模型为"FM 当验证器"标记出的奖励黑客风险是同一个，只是被搬到了"FM 当模拟器"上。第三，**吞吐天花板**——瓶颈是机器，不是模型，所以这个闭环只有在 *批量* 主动学习（每轮许多多样的实验）以及对"一笔预算实际能买到多少真实周期"的残酷诚实之下才划得来。第四，**机器速度下的多重比较**——一个自主闭环能在人类还没注意到它在 p-hacking 之前就生成上百个看着像样的命中，这与 CellVoyager 批评针对自主分析提出的危害是同一个，现在还接上了一只机械臂。而在这四条之下，坐着那条显而易见的：自主生物学需要生物防护，以及对闭环被允许实体制造出什么的人类把关。

## 它的去向

终点是闭环可达性实验室。基础模型提出哪些细胞状态应当可工程化；自驱动实验室去尝试它们；失败与成功把已实现 / 可工程化 / 禁区的边界重画成一个被测量的对象，而非一个主张。这才是最终让那道可生存性分层 *在世界里可证伪* 而非仅在计算机里可证伪的东西——你不去争论一个状态是否是禁区，你看着机器一轮接一轮地造它不出来，直到那道边界成为经验性的。没有世界模型的智能体是个盲眼规划者；没有预言机的基础模型是个笃定的猜测者；没有策略的机器是一次昂贵的随机游走。把它们放进一个闭环，你就得到了这个领域自第一篇虚拟细胞展望以来一直绕着转的那个东西：一个提出生物学、检验它、并从犯错中学习的系统。引擎已经造好。剩下的，是像化学早已在分子上闭合闭环那样，在细胞上把闭环闭合。
