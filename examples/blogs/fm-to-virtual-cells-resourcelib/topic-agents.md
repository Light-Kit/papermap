The agentic thread is the most commercially fundable lane in 2026. Three roles agents play: (1) agents that BUILD models — vcHarness (end-to-end virtual cell construction); (2) agents that ANALYSE data — CellVoyager (single-cell analysis with the FM as substrate); (3) agents that REASON OVER a virtual cell — rBio (verifier-style); (4) agents that USE biology FMs in workflow — Owkin Pathology Explorer (pathology agent inside Claude), PathChat-2/DX. DeepRare (Nature 2026) is the clinical proof point: multi-agent LLM diagnosis beating physicians on rare disease. VC-World (arXiv 2025) is the anti-black-box alternative: white-box LLM-driven simulator with stepwise mechanistic reasoning, SOTA on drug-perturbation benchmarks. Li-agentic 2026 is the position paper behind the agentic-FM intersection. DrugReflector 2026 is the AI drug-screening agent picked up in industry coverage.
