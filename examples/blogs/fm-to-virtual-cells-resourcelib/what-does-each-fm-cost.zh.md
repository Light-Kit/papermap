## 一句话头条

截至 2026 年 5 月，跑完一次前沿生物学 FM 的完整训练，要 **2 万到 500 万美元以上**。Geneformer V2-104M_CLcancer（公开披露 1.7 万美元）是我们这张矩阵里成本完整披露的最便宜的锚点模型；Evo2（500 万美元，2,048 块 H100 跑好几个月）是披露过的最贵的。**中间各档全都存在，只是大多数实验室披露得没有 NVIDIA 联名的那些细**——下面那条「披露程度和赞助挂钩」的规律会讲到。

## 这张矩阵让四条规律浮了出来

### 1. 披露程度和硬件赞助挂钩

每一条 ✅ 已披露的条目，背后都有一段 NVIDIA / DGX-Cloud / TPU 的联名作者关系，或者框架层面的合作。**没有**硬件联合营销方的学术实验室，披露得少得多——这是结构性的，不是随机的。披露这件事本身，就是一种联合营销的产物。

| ✅ 已披露 | 赞助方 |
|---|---|
| Geneformer V2 | NVIDIA BioNeMo（Theodoris 组） |
| Virchow2 | Paige 的 V100 机群 |
| Evo2 | NVIDIA + DGX Cloud（Arc + NVIDIA 论文） |
| ESM-3 | EvolutionaryScale 的 H100 机群 |
| AlphaGenome | Google TPU v3（部分） |

| ❌ 未知 | 实验室 |
|---|---|
| scGPT | Bo Wang 组（无硬件联合营销方） |
| UNI / UNI2-h | Mahmood 组 |
| UCE | Leskovec + Quake / Stanford |
| STATE | Arc Institute（这一个没有联名作者） |

### 2. 2 万美元的地板——最便宜、可复现的 sc-FM 训练是哪一个？

**Geneformer V2-104M_CLcancer**：64× A100 80GB × 4 天 8 小时 = 6,656 A100 小时 = 按需价 **16,640 美元**。再加上那一步在 1,400 万癌症细胞上做持续预训练的约 3,000 美元。**全算下来 2 万美元，完整披露，NVIDIA BioNeMo 配方公开。** 这是唯一一例已发表的学术 sc-FM 训练，小实验室能照着一份完整配方把成本核验出来。

2 万以下的，要么是概念验证（Lewis & Zueco，250 美元），要么没披露。2 万以上*且同时披露*的，无一不需要一个硬件合作方。

### 3. 500 万美元的天花板——披露过的最贵生物学 FM 是哪个？

**Evo2 (40B)**：2,048× H100，跑在 NVIDIA DGX Cloud（AWS）上，「好几个月」。约 200 万 H100 小时，按需价约 500 万。Evo2 的 *Nature* 论文明确披露：「比 AlphaFold 多约 150 倍算力，约为 ESM-3 的 2 倍 FLOPs」——这是 2025–2026 年发表过的最干净的一组跨模型算力比值。

**ESM-3 (98B)**：在 *Science* 里以单个数字披露为 1.07 × 10²⁴ FLOPs。从业者可以拿任意等效的 FLOPs 预算，在自己偏好的硬件上做成本对照。EvolutionaryScale 把这个数放出来，作为全领域的算力参照点。

### 4. 病理 FM 是中间档（7.5 万–34 万），而且比 sc-FM 披露得多

病理这一家族是*透明度上的异类*。Virchow2 / Virchow2G 公布了硬件数量加患者层面的来源出处。UNI / UNI2-h 则把算力藏在「大量 A100 小时」后面。这道分水岭是机构性的：Paige（商业衍生公司，需要硬件披露来换取供应商信任）对 Mahmood 组（学术，没有联合营销方）。

## 这对小实验室的规划意味着什么

**预训练这件事，你竞争不了。** 500 万美元的训练集中在全球约 5 家机构手里。哪怕只是 17 万（Virchow2），也得有一家商业病理实验室才凑得齐的 V100 机群。没有 NVIDIA 联名的学术单细胞实验室，是没法照着已发表配方复现一次前沿 sc-FM 训练的。

**但小实验室的打法本来就不是预训练。** 它是：

- **赛道 1——冻结嵌入的活儿**（0–500 美元）：只做推理，1× 消费级 GPU，几周搞定。
- **赛道 2——PEFT / LoRA / 适配器**（500–5,000 美元）：1–8× A100 跑几天，2–4 个月。适配器是你的贡献，预训练是租来的。
- **赛道 3——领域专用的小 FM**（1 万–5 万美元）：把 Geneformer V2-104M_CLcancer 那套打法搬到你自己的疾病领域。6–12 个月。
- **赛道 4——复现 / 批评**（0–2,000 美元）：推理 + 线性回归。Ahlmann-Eltze 用不到 2,000 美元，就把整张 sc-FM 扰动预测排行榜给报废了。

完整的 9 条赛道按预算分档的概览，见[小实验室——该造什么](small-labs-what-to-build.md)。

## 怎么读一份算力声明

看到一篇论文声称某个 FM 训练花了 X，去核查这几点：

1. **是已披露、估算的，还是未知的。** 作者有没有把 GPU 数量、小时数、硬件型号说清楚？如果只说了「大量 A100 小时」，那就是未知——信封背面的估算，误差带很宽。
2. **是一次跑成，还是跑了很多次？** 前沿训练常常要重试。报出来的成本一般是那次*成功*的跑。论文要是说「我们在 64 块 A100 上训了一周」，却不提之前失败了多少次，那真实总成本得乘上 2–10 倍。
3. **推理成本 vs 训练成本。** 有些论文把两者混在一起。训练是一次性的、贵；推理是按次的、便宜。对赛道 1（冻结嵌入）来说，只有推理才重要。
4. **预训练成本 vs 微调成本。** 论文要是报「训练花了 X」，得查清这是只算预训练，还是连下游微调一起算了。微调通常比预训练便宜 10–100 倍。
5. **软件栈。** NVIDIA BioNeMo、JAX、MosaicML 等等，每 FLOP 的成本画像各不相同。一篇 NVIDIA 联名、带 BioNeMo 配方的论文，是最可复现的那种披露范式。
