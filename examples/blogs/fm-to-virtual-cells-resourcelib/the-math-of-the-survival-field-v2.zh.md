> *这是[生存场数学入门](what-disappears-conditional-viability-v3.md)的第二版——v1 从头教了那几样经典对象，结果被回了一句中肯的话：「这不就是老数学吗，ML 在哪？」这一版正面接住它，办法是把模型拆成模块、为每块各挑一件工具。要看「为什么」，去读[v3 模型那篇](what-disappears-conditional-viability-v3.md)；这一篇讲的是「怎么搭」。它是[可达性 / 可生存性方案](cell-state-reachability-as-viability-theory.md)的一部分，建在[扩散 / 密度主干](diffusion-and-flow-matching-for-virtual-cells.md)之上，本质上是个[因果](causal-models-fm-and-vc.md)对象，其不确定性经由[惊讶 / 不确定性信号](surprise-and-uncertainty-in-cell-fms.md)馈入[agent + FM + 机器闭环](agent-fm-machine-closed-loop.md)。*

## 问错了，和问对了

「这就是老数学——要不要加点 ML？」这问法本身就错了，因为它把数学和 ML 当成了可以互换的两样东西。它们不是。**经典数学供的是归纳偏置**——一个约束、一句可识别性陈述、一条外推保证；**现代 ML 供的是函数逼近**——从数据里学出来的、灵活的高维拟合。一个好模型，是让两者各干各擅长的活，而工程的精髓，就在它们之间那道**缝**。所以问题应该这么问：把任务拆成模块，对每一块都掂量一下——它吃紧的那部分究竟是个*约束*（那就找数学），还是个*逼近*（那就找 ML），然后把接口对齐。这篇就是干这件事。

## 给定的，和要求的

**给定。** [Tahoe-100M](https://www.biorxiv.org/content/10.1101/2025.02.20.639398v3.full)：约 1,100 种药 × 50 个细胞系 × 一张剂量网格上的对照与给药单细胞群体，以成分计数的形式给出，只有 24 h 的一张快照。一个冻结的细胞状态编码器 [Tahoe-x1](https://doi.org/10.1101/2025.10.23.683759)，给出 *z* ∈ ℝᵏ。一个冻结的药物结构编码器 [MoLFormer](https://arxiv.org/abs/2106.09553)，给出 *c_g*。一个标量剂量 *d*。

**要求。** 一个生存场 π(*z*\|*g*,*d*) 加上一套幸存者传输，二者合起来要能：（a）**把死亡和状态迁移劈开**，（b）**推广到没见过的药和剂量**，（c）守住**成分可识别性**，（d）为实验选择那个环节吐出**校准过的不确定性**。

## 任务图

> 编码器 (M0) → 条件化 (M1) → { 死亡分支 (M2) ‖ 移动分支 (M3) } → 计数似然 + 可识别性 (M4) → 一个目标 (M5) → 不确定性 / 采集 (M6) → 与基线对比的评估 (M7)。

M2 和 M3 这两条分支是核心，而整套设计最漂亮的一句话就在这——现代非平衡传输能让它们成为**同一个网络的两个输出**，因为「区分死亡和移动」所依赖的那套几何，恰恰就是 flow matching 在估计的那套几何。一块块来看：

## M0 — 编码器：状态与结构

**任务。** *z* = 细胞状态，*c_g* = 药物，外加一个剂量表示。
**经典数学。** PCA 或线性隐变量；ECFP 分子指纹；对 log 剂量做样条 / 基展开。
**现代 ML。** 冻结的 [Tahoe-x1](https://doi.org/10.1101/2025.10.23.683759) 和 [MoLFormer](https://arxiv.org/abs/2106.09553)；万一冻结的隐空间把跟存活相关的轴给丢了（这风险是实打实的），就插几个小 [adapter](https://arxiv.org/abs/1902.00751)，只微调这些。
**怎么选。** 冻结 FM + 可选 adapter——前端既保持模块化、又便宜，同时一旦评估有需要，隐空间还能补上存活感知。
**接口。** 把 (*z*, *c_g*, *d*) 交给 M1。

## M1 — 条件化：那条共享的摊销主干

**任务。** 把 (*c_g*, *d*) 变成一个调制信号，去专门化下游每一个头，好让模型能推广到*没见过*的药和剂量，而不是把见过的孔位死记下来。
**经典数学。** 把协变量拼接起来；基特征。
**现代 ML。** [FiLM](https://arxiv.org/abs/1709.07871) 的逐特征 (γ, β) 调制；从条件向量生成权重的[超网络](https://arxiv.org/abs/1609.09106)；交叉注意力 / adapter。
**怎么选。** 默认用 FiLM（便宜、验证过），某个药需要强专门化时再上超网络。
**接口——最要紧的那道。** 算出*一个*条件向量 **h = f(c_g, d)**，用它同时驱动死亡分支 (M2) **和**移动分支 (M3)。一条主干、两个头：正是这条共享的 **h**，让整个模型是个连贯的整体，而不是两个硬钉在一起的子模型。

## M2 — 死亡分支（危险率 / 生存）

**任务。** π(*z*\|*g*,*d*) = exp(−τ·λ)，关于剂量单调。
**经典数学。** Cox 比例风险、竞争风险，以及可靠性工程里的**加速寿命试验**——把剂量当应力，配一条能*外推*的逆幂律 / Arrhenius 加速律；放射生物学的线性二次曲线就是它的标量特例。
**现代 ML。** 深度危险率模型——[DeepSurv](https://arxiv.org/abs/1606.00931)、[DeepHit](https://ojs.aaai.org/index.php/AAAI/article/view/11842)（竞争风险，不假设 PH）、[SODEN](https://arxiv.org/abs/2008.08637)（神经 ODE 危险率）、[Deep Survival Machines](https://arxiv.org/abs/2003.01176)——再加上那招关键的：用 [UMNN](https://arxiv.org/abs/1908.05164)、[受约束单调网络](https://arxiv.org/abs/2205.11775)或[深度格网络](https://arxiv.org/abs/1709.06680)，**从架构上把剂量单调性钉死**。
**怎么选——一次数学 / ML 的融合。** 保留 ALT 的函数形式，让网络去学它的各个部件，并把剂量单调性*担保*下来：

> log λ(*z*,*g*,*d*) = log λ₀(*z*) + monotone_net( log *d* ; **h** )，

其中 λ₀ 是 *z* 上的一个 MLP，`monotone_net` 是一个 UMNN，其输出在构造上就关于剂量单调。经典骨架（ALT、exp(−H)）、神经肌肉（灵活的头）、硬约束（任何药在更高剂量下都不会变得更安全）——这三样，黑箱一样都不会白送给你。
**接口。** π 在传输之前对对照样本重新加权——也就是下一模块里的 Fisher–Rao / 反应那一侧。

## M3 — 移动分支（传输）

**任务。** 把幸存者从对照状态搬到给药状态。
**经典数学。** 最优传输——Monge/Kantorovich、Brenier 的「映射 = 凸势的梯度」、熵正则的 [Sinkhorn](https://arxiv.org/abs/1306.0895)，以及允许质量生灭的**非平衡 OT**（[Wasserstein–Fisher–Rao / Hellinger–Kantorovich](https://arxiv.org/abs/1506.06430)）。
**现代 ML。** 工具箱跑得最快的就是这一块。静态神经 OT：[ICNN Monge 映射](https://arxiv.org/abs/1908.10962)、[Monge gap](https://arxiv.org/abs/2302.04953) 正则项、可微的 [Sinkhorn 散度](https://arxiv.org/abs/1706.00292)。动态、生成式的传输：[flow matching](https://arxiv.org/abs/2210.02747)、[OT-CFM](https://arxiv.org/abs/2302.00482)，以及 [Schrödinger bridges](https://arxiv.org/abs/2106.01357)（[DSBM](https://arxiv.org/abs/2303.16852)、[[SF]²M](https://arxiv.org/abs/2307.03672)）——一座 Schrödinger bridge 说白了就是*动态的熵正则 OT*，所以它和 v1 教过的是同一个对象，只是被做成了可学的 SDE。还有——正对我们这个问题的——**非平衡 / 生长感知**那一族：[Unbalanced Diffusion Schrödinger Bridge](https://arxiv.org/abs/2306.09099)（带杀灭 / 新生项，已在*药物诱导的细胞死亡*上演示过）、[CytoBridge / 非平衡平均场 SB](https://arxiv.org/abs/2505.11197)，以及 [Joint Velocity–Growth Flow Matching](https://arxiv.org/abs/2505.13413)，它通过半松弛 OT 把一个位移场*和*一个逐细胞的生长 / 死亡率一起学出来。[Metric flow matching](https://arxiv.org/abs/2405.14780) 则负责把传输后的幸存者按在数据流形上。
**怎么选——以及那次统一。** 用一个**生长感知流 / 非平衡 Schrödinger bridge**。它的标志性特征，就是 v1 那句结论的可操作版：任何演化都能劈成一个**传输速度**（质量往哪挪）加一个**生长率**（质量冒出来多少、消失多少）。于是网络把*两者都*吐出来——而那个**生长率输出*就是*死亡场 π**，那个速度*就是*状态迁移。M2 和 M3 不再是两个模块，而是同一个生长感知流的两个头，由同一条 **h** 条件化。
**接口。** 这是整个模型的中枢缝合处：WFR 几何*定义*了生物学要的那道死亡 / 传输劈分，而 flow matching 只是它那个可扩展的估计器。经典数学说的是这道分解*是什么*；ML 说的是怎么在 1 亿细胞的尺度上*把它拟合出来*。

## M4 — 计数似然与可识别性

**任务。** 把预测的给药分布和观测到的成分计数对上，并直面那个常数。
**经典数学。** 一个 Dirichlet-multinomial / 成分似然；那条**可识别性条款**——成分计数只能把 π 钉到一个逐条件的乘性常数那么准（占据率 / Elkan–Noto / Heckman 常数）。这是个*约束*，再多 ML 也化不开它。
**现代 ML。** 给计数过程配一个负二项 / ZINB 解码器，路子如 [scVI](https://doi.org/10.1038/s41592-018-0229-2) 或 [ZINB-WaVE](https://doi.org/10.1038/s41467-017-02554-5)；评估群体偏移时用 [scCODA](https://doi.org/10.1038/s41467-021-27150-6) / [sccomp](https://doi.org/10.1073/pnas.2203828120)，但只是辅助，不当核心。
**怎么选——数学领头。** 给计数配一个 NB 似然，再加一个低剂量校准先验（接近零剂量时存活 ≈ 1），或者一个 spike-in / 绝对计数锚来钉死那个常数。把诚实的边界说在前头：没有锚的话，能识别的只是跨状态、跨剂量的*相对*存活。
**接口。** 给目标函数供上数据拟合那一项。

## M5 — 一个可微的目标

**任务。** 把这一切端到端地拟合出来。
**复合损失** = 一个生长感知 flow-matching / Schrödinger-bridge 传输项 + 一个非平衡的质量松弛（死亡）项 + NB 成分似然 + 一个 Monge-gap / 熵正则项（剂量单调性是*白送*的，已经焊进 M2 的架构里了）。在 Tahoe-100M 上按孔位做小批量 SGD 训练，**留出整分子和整剂量档**，让泛化能被诚实地检验。可微的 [Sinkhorn](https://arxiv.org/abs/1706.00292) 和免仿真的 [flow-matching](https://arxiv.org/abs/2210.02747) 损失，让这一切都收在一次反向传播里。
**接口。** 一张图、一条共享的 **h** 主干、一个优化器。

## M6 — 不确定性与采集

**任务。** 在没见过的 (*g*,*d*) 上给出校准过的不确定性 → 挑下一批实验。
**经典数学。** 高斯过程；[conformal 预测](https://arxiv.org/abs/2107.07511)，给出免分布、有限样本的保证。
**现代 ML。** [深度集成](https://arxiv.org/abs/1612.01474)、[MC-dropout](https://arxiv.org/abs/1506.02142)、[证据深度学习](https://arxiv.org/abs/1806.01768)。
**怎么选。** 用深度集成给预测不确定性 + conformal 给查询把关——这个采集信号会喂给那个[闭环](agent-fm-machine-closed-loop.md)。

## M7 — 评估与基线

留出的孔位（内插）、没见过的分子（那个吃重的测试）、没见过的剂量（ALT 外推那个测试），以及 BH3 profiling 相关性（一个正交的机理锚）。基线，按各自能覆盖的范围排：[Waddington-OT](https://doi.org/10.1016/j.cell.2019.01.006) 和 [moscot](https://doi.org/10.1038/s41586-024-08453-2)（能给死亡率，但逐条件、不泛化）；[CellOT](https://doi.org/10.1038/s41592-023-01969-x)（神经 OT 响应，质量守恒）；[chemCPA](https://arxiv.org/abs/2204.13545) 和 [CellFlow](https://doi.org/10.1101/2025.04.11.648220)（药物条件化的泛化，但给的是存活者的表达，不是存活本身）；Arc 的 [State](https://doi.org/10.1101/2025.06.26.661135) 模型（一个强的摊销式对照）。

## 整个模型，一张表收齐

| 模块 | 任务 | 经典数学工具 | 现代 ML 工具 | 我们的选择 |
|---|---|---|---|---|
| M0 编码器 | 状态、药、剂量 → 向量 | PCA、指纹、样条 | Tahoe-x1 + MoLFormer、adapter | 冻结 FM + 可选 adapter |
| M1 条件化 | 在 (g,d) 上摊销 | 协变量拼接 | FiLM、超网络、adapter | FiLM → 共享 **h** |
| M2 死亡 | π = exp(−τλ)，剂量单调 | Cox / 竞争风险 / ALT | DeepHit、SODEN、UMNN | ALT 形式 + 单调网络 |
| M3 移动 | 传输幸存者 | OT / 非平衡 OT (WFR) | flow matching、SB、**生长感知 FM/UDSB** | 生长感知流（死亡 = 它的生长头） |
| M4 似然 | 拟合计数 + 钉常数 | Dirichlet-multinomial、可识别性 | scVI NB/ZINB | NB + 校准锚 |
| M5 目标 | 端到端拟合 | 熵正则 UOT 散度 | Sinkhorn/flow-matching 损失 | 可微复合损失 |
| M6 不确定性 | 校准、采集 | GP、conformal | 深度集成、证据式 | 集成 + conformal |
| M7 评估 | 证伪、对比 | 留出式设计 | WOT/moscot/CellOT/chemCPA/CellFlow/State | 四个留出测试 + BH3 |

## 它们是怎么咬合的——再把那道平滑的缝说一遍

从上往下读，它就是一张可微的图：**冻结编码器**产出 (*z*, *c_g*, *d*)；**一个条件向量 h(c_g,d)** 调制**一个生长感知流网络**，后者吐出一个**速度**（幸存者传输）和一个**生长率**（它变成生存场 π，M2 的 ALT / 单调结构就焊在这个头里）；那份（被 π 削过、又传输过的）对照样本，在 **NB 似然**下与观测到的给药计数相比；整套东西用一个**复合 UOT + 似然损失**拟合，再**集成**起来给出不确定性。这些连接之所以干净，只有一个原因：**经典数学和 ML 描述的是同一批对象，只是分辨率不同。** 生存 = exp(−H)、WFR 那道死亡 / 传输劈分、剂量单调、成分可识别性，这些是*约束*；深度编码器、FiLM 条件化、单调神经危险率、flow matching，这些是*逼近器*；而非平衡传输，正是两者交汇的那个点——几何给分解起了名字，网络把它估出来。

## 纯 ML 会在哪里跑偏

诱人的捷径，是把这一身结构全扔了，训一个大大的、给治疗后细胞做条件生成的生成器。它会管用——而且会把这个领域现有的窟窿原样复现：你预测的将是*存活者的表达*（CPA、[scGen](https://doi.org/10.1038/s41592-019-0494-8)、CellFlow 早就做得不错的东西），却悄无声息地把死亡 / 传输劈分、剂量单调、可识别性条款全丢了。这三样不是装饰，它们是这东西之所以是个*生存*模型、而不是又一个扰动预测器的全部理由。所以纪律是定死的：**地图和条件化交给现代 ML，约束和可识别性交给经典数学**——而非平衡传输那道缝，正是让两者合成一个模型、而不是两个模型的关键。

## 一句话结论

老数学是骨架——约束、可识别性、外推保证。现代 ML 是肌肉——flow matching、摊销式条件化、单调神经危险率。模型就是那个**接合处**，而接头处正是 Wasserstein–Fisher–Rao 那道劈分：一个生长感知流的生长头*就是*生存场，它的速度头*就是*状态迁移。一个网络、条件化一次、训练一次，终于把现代工具箱整个对准，去建模*谁活了下来*。
