> *[存活场数学入门](what-disappears-conditional-viability-v3.md)的第 2 版——v1 从零教起那些经典对象；中肯的回应是"这不过是老数学，ML 在哪里？"本版本以恰当的方式回答它，办法是把模型拆成模块、为每个模块挑选一件工具。原因请读 [v3 模型随笔](what-disappears-conditional-viability-v3.md)；这一篇讲的是工程。本文是[可达性 / 存活性方案](cell-state-reachability-as-viability-theory.md)的一部分，建立在[扩散 / 密度主干](diffusion-and-flow-matching-for-virtual-cells.md)之上，是一个[因果](causal-models-fm-and-vc.md)对象，其不确定性通过[惊奇 / 不确定性信号](surprise-and-uncertainty-in-cell-fms.md)馈入 [agent + FM + machine 闭环](agent-fm-machine-closed-loop.md)。*

## 错误的问题与正确的问题

"这不过是老数学——我们要不要加点 ML？"是个错误的问题，因为它把数学和 ML 当成了替代品。它们不是。**经典数学提供归纳偏置**——一个约束、一句可识别性陈述、一项外推保证——而**现代 ML 提供函数逼近**——从数据学到的灵活、高维的拟合。一个好模型让两者各司其所，而工程就是它们之间的那道 *接缝*。所以正确的问题是：把任务拆成模块，对每个模块去问，承重的那部分究竟是一个 *约束*（伸手去拿数学）还是一个 *逼近*（伸手去拿 ML），然后让接口彼此对齐。这就是本文。

## 给定项与目标

**给定项。** [Tahoe-100M](https://www.biorxiv.org/content/10.1101/2025.02.20.639398v3.full)：约 1,100 种药物 × 50 个细胞系 × 一个剂量网格上的对照与药物处理单细胞群体，以成分计数形式给出，一个 24 h 快照。一个冻结的细胞状态编码器 [Tahoe-x1](https://doi.org/10.1101/2025.10.23.683759)，给出 *z* ∈ ℝᵏ。一个冻结的药物结构编码器 [MoLFormer](https://arxiv.org/abs/2106.09553)，给出 *c_g*。一个标量剂量 *d*。

**目标。** 一个存活场 π(*z*\|*g*,*d*) 与一个存活者传输，二者共同（a）**把死亡从状态转移中分离出来**，（b）**泛化到未见过的药物与剂量**，（c）尊重**成分可识别性**，以及（d）为实验选择回路给出**校准好的不确定性**。

## 任务图

> 编码器 (M0) → 条件化 (M1) → { 死亡分支 (M2) ‖ 移动分支 (M3) } → 计数似然 + 可识别性 (M4) → 一个目标 (M5) → 不确定性 / 采集 (M6) → 与基线对比的评估 (M7)。

M2 与 M3 这两条分支是核心，并且——这是整个设计的妙处所在——现代非平衡传输让它们能成为**一个网络的两个输出**，因为把死亡与移动分离开来的几何，正是 flow matching 所估计的同一种几何。逐模块来看：

## M0——编码器：状态与结构

**职责。** *z* = 细胞状态，*c_g* = 药物，外加一个剂量表示。
**经典数学。** PCA 或一个线性隐变量；ECFP 分子指纹；对 log-dose 的样条 / 基展开。
**现代 ML。** 冻结的 [Tahoe-x1](https://doi.org/10.1101/2025.10.23.683759) 与 [MoLFormer](https://arxiv.org/abs/2106.09553)；如果冻结的隐空间丢掉了与存活相关的轴（这是个真实风险），就插入小型 [adapters](https://arxiv.org/abs/1902.00751)，只微调它们。
**抉择。** 冻结 FM + 可选 adapter——既保住模块化、廉价的前端，又能在评估有此要求时让隐空间变得具备存活感知。
**接缝。** 向 M1 发出 (*z*, *c_g*, *d*)。

## M1——条件化：共享的摊销主干

**职责。** 把 (*c_g*, *d*) 转化为一个调制信号，专门化每一个下游 head，使模型泛化到 *未见过的* 药物与剂量，而非死记已见过的孔。
**经典数学。** 拼接协变量；基特征。
**现代 ML。** [FiLM](https://arxiv.org/abs/1709.07871) 的逐特征 (γ, β) 调制；从条件向量生成权重的[超网络](https://arxiv.org/abs/1609.09106)；交叉注意力 / adapter。
**抉择。** 默认用 FiLM（廉价、已验证），若某种药物需要强力专门化则用超网络。
**接缝——最重要的那个。** 计算一个 *单一* 的条件向量 **h = f(c_g, d)**，并用它去驱动死亡分支 (M2) 与移动分支 (M3) **二者**。一条主干、两个 head：正是那个共享的 **h** 让模型成为一个连贯的整体，而不是两个钉在一起的子模型。

## M2——死亡分支（危险率 / 存活）

**职责。** π(*z*\|*g*,*d*) = exp(−τ·λ)，对剂量单调。
**经典数学。** Cox 比例危险、竞争风险，以及可靠性工程的 **加速寿命试验**——把剂量当作应力，配上一个能 *外推* 的反幂律 / Arrhenius 加速；放射生物学的线性二次曲线是其标量特例。
**现代 ML。** 深度危险率模型——[DeepSurv](https://arxiv.org/abs/1606.00931)、[DeepHit](https://ojs.aaai.org/index.php/AAAI/article/view/11842)（竞争风险，无 PH 假设）、[SODEN](https://arxiv.org/abs/2008.08637)（神经-ODE 危险率）、[Deep Survival Machines](https://arxiv.org/abs/2003.01176)——以及关键的把戏：**用架构来强制剂量单调性**，借助 [UMNN](https://arxiv.org/abs/1908.05164)、一个[受约束单调网络](https://arxiv.org/abs/2205.11775)，或一个[深度格网络](https://arxiv.org/abs/1709.06680)。
**抉择——一次数学 / ML 的融合。** 保留 ALT 的函数形式，用网络学习它的各个部件，并让剂量单调性得到 *保证*：

> log λ(*z*,*g*,*d*) = log λ₀(*z*) + monotone_net( log *d* ; **h** )，

其中 λ₀ 是作用于 *z* 上的一个 MLP，`monotone_net` 是一个 UMNN，其输出按构造对剂量单调。经典骨架（ALT、exp(−H)）、神经肌肉（灵活的 head）、硬约束（没有哪种药物在更高剂量下反而更安全）——这些都不是黑箱能白送给你的。
**接缝。** π 在传输之前对对照样本重新加权——这是下一模块中 Fisher–Rao / 反应的那一侧。

## M3——移动分支（传输）

**职责。** 把存活者从对照状态搬到处理状态。
**经典数学。** 最优传输——Monge/Kantorovich、Brenier 的"映射 = 一个凸势的梯度"、熵正则的 [Sinkhorn](https://arxiv.org/abs/1306.0895)，以及允许质量产生 / 消灭的 **非平衡 OT**（[Wasserstein–Fisher–Rao / Hellinger–Kantorovich](https://arxiv.org/abs/1506.06430)）。
**现代 ML。** 这正是工具箱迭代最快的地方。静态神经 OT：[ICNN Monge 映射](https://arxiv.org/abs/1908.10962)、[Monge gap](https://arxiv.org/abs/2302.04953) 正则项、可微的 [Sinkhorn 散度](https://arxiv.org/abs/1706.00292)。动态、生成式传输：[flow matching](https://arxiv.org/abs/2210.02747)、[OT-CFM](https://arxiv.org/abs/2302.00482)，以及 [Schrödinger bridges](https://arxiv.org/abs/2106.01357)（[DSBM](https://arxiv.org/abs/2303.16852)、[[SF]²M](https://arxiv.org/abs/2307.03672)）——一个 Schrödinger bridge 不过是 *动态熵 OT*，所以它就是 v1 教过的同一个对象，被做成了一个可学习的 SDE。而且——直接针对我们的问题——还有 **非平衡 / 生长感知** 这一族：[Unbalanced Diffusion Schrödinger Bridge](https://arxiv.org/abs/2306.09099)（带消灭 / 产生项，已在 *药物诱导的细胞死亡* 上演示）、[CytoBridge / 非平衡平均场 SB](https://arxiv.org/abs/2505.11197)，以及 [Joint Velocity–Growth Flow Matching](https://arxiv.org/abs/2505.13413)，后者通过半松弛 OT 同时学习一个位移场 *和* 一个每细胞的生长 / 死亡率。[Metric flow matching](https://arxiv.org/abs/2405.14780) 让被传输的存活者留在数据流形上。
**抉择——以及那次统一。** 使用一个 **生长感知 flow / 非平衡 Schrödinger bridge**。它的标志性特征正是 v1 那句妙语的可操作化：任何演化都分解为一个 **传输速度**（质量往哪里移动）外加一个 **生长率**（有多少质量出现或消失）。所以网络同时发出 *二者*——而那个 **生长率输出 *就是* 死亡场 π**，速度 *就是* 状态转移。M2 与 M3 不再是两个模块，而成为同一个生长感知 flow 的两个 head，受同一个 **h** 的条件作用。
**接缝。** 这是整个模型的中心接缝：WFR 几何 *定义* 了生物学所需的那个死亡 / 传输分解，而 flow matching 仅仅是它的可扩展估计器。经典数学说出这个分解 *是什么*；ML 说出 *如何* 在 1 亿细胞的规模上去拟合它。

## M4——计数似然与可识别性

**职责。** 把预测的处理分布与观测的成分计数作比较，并直面那个常数。
**经典数学。** 一个 Dirichlet-multinomial / 成分似然；**可识别性条款**——成分计数只能把 π 钉到一个逐条件的乘性常数（占用度 / Elkan–Noto / Heckman 常数）以内。这是一个 *约束*，再多的 ML 也消解不了它。
**现代 ML。** 一个仿照 [scVI](https://doi.org/10.1038/s41592-018-0229-2) 或 [ZINB-WaVE](https://doi.org/10.1038/s41467-017-02554-5) 的负二项 / ZINB 解码器来处理计数过程；[scCODA](https://doi.org/10.1038/s41467-021-27150-6) / [sccomp](https://doi.org/10.1073/pnas.2203828120) 用于评估群体偏移，而非作为核心。
**抉择——数学领衔。** 对计数用一个 NB 似然，外加一个低剂量校准先验（在零剂量附近存活率 ≈ 1）或一个掺标 / 绝对计数锚点来钉住那个常数。诚实地说出极限：没有锚点，只有跨状态、跨剂量的 *相对* 存活率是可识别的。
**接缝。** 提供目标函数中的数据拟合项。

## M5——一个可微的目标

**职责。** 端到端拟合这一切。
**复合损失** = 一个生长感知的 flow-matching/Schrödinger-bridge 传输项 + 一个非平衡的质量松弛（死亡）项 + NB 成分似然 + 一个 Monge-gap/熵正则项（剂量单调性是 *免费的*，已烘焙进 M2 的架构）。在 Tahoe-100M 上以孔为单位、用 minibatch SGD 训练，并 **留出整分子与整个剂量水平**，从而诚实地检验泛化。可微的 [Sinkhorn](https://arxiv.org/abs/1706.00292) 与免仿真的 [flow-matching](https://arxiv.org/abs/2210.02747) 损失让这一切保持在一次反向传播之内。
**接缝。** 一张图、一条共享的 **h** 主干、一个优化器。

## M6——不确定性与采集

**职责。** 在未见过的 (*g*,*d*) 上给出校准好的不确定性 → 选择下一批实验。
**经典数学。** 高斯过程；用于无分布、有限样本保证的 [conformal prediction](https://arxiv.org/abs/2107.07511)。
**现代 ML。** [Deep ensembles](https://arxiv.org/abs/1612.01474)、[MC-dropout](https://arxiv.org/abs/1506.02142)、[evidential deep learning](https://arxiv.org/abs/1806.01768)。
**抉择。** 用深度集成给出预测不确定性 + 用 conformal 来把关查询——这就是馈入[闭环](agent-fm-machine-closed-loop.md)的采集信号。

## M7——评估与基线

留出的孔（插值）、未见过的分子（承重的那项测试）、未见过的剂量（ALT 外推测试），以及 BH3-profiling 相关性（一个正交的机制锚点）。各基线，按它们所覆盖的内容：[Waddington-OT](https://doi.org/10.1016/j.cell.2019.01.006) 与 [moscot](https://doi.org/10.1038/s41586-024-08453-2)（死亡率，但逐条件、不泛化）；[CellOT](https://doi.org/10.1038/s41592-023-01969-x)（神经 OT 响应，质量守恒）；[chemCPA](https://arxiv.org/abs/2204.13545) 与 [CellFlow](https://doi.org/10.1101/2025.04.11.648220)（药物条件化的泛化，但是存活者的表达，而非存活本身）；Arc 的 [State](https://doi.org/10.1101/2025.06.26.661135) 模型（强力的摊销式对照者）。

## 整个模型一张表

| 模块 | 职责 | 经典数学工具 | 现代 ML 工具 | 我们的抉择 |
|---|---|---|---|---|
| M0 编码器 | 状态、药物、剂量 → 向量 | PCA、指纹、样条 | Tahoe-x1 + MoLFormer、adapter | 冻结 FM + 可选 adapter |
| M1 条件化 | 对 (g,d) 摊销 | 协变量拼接 | FiLM、超网络、adapter | FiLM → 共享 **h** |
| M2 死亡 | π = exp(−τλ)，剂量单调 | Cox / 竞争风险 / ALT | DeepHit、SODEN、UMNN | ALT 形式 + 单调网络 |
| M3 移动 | 传输存活者 | OT / 非平衡 OT (WFR) | flow matching、SB、**生长感知 FM/UDSB** | 生长感知 flow（死亡 = 它的生长 head） |
| M4 似然 | 拟合计数 + 钉住常数 | Dirichlet-multinomial、可识别性 | scVI NB/ZINB | NB + 校准锚点 |
| M5 目标 | 端到端拟合 | 熵 UOT 散度 | Sinkhorn/flow-matching 损失 | 复合可微损失 |
| M6 不确定性 | 校准、采集 | GP、conformal | 深度集成、evidential | 集成 + conformal |
| M7 评估 | 证伪、比较 | 留出设计 | WOT/moscot/CellOT/chemCPA/CellFlow/State | 四项留出测试 + BH3 |

## 它如何相连——那道平滑接缝，再说一遍

从上读到下，它是一张单一的可微图：**冻结编码器** 产出 (*z*, *c_g*, *d*)；**一个条件向量 h(c_g,d)** 调制 **一个生长感知 flow 网络**，后者发出一个 **速度**（存活者传输）与一个 **生长率**（它成为存活场 π，M2 的 ALT/单调结构被接入那个 head）；（被 π 削减、被传输的）对照样本在一个 **NB 似然** 下与观测到的处理计数作比较；整个东西用一个 **复合 UOT + 似然损失** 来拟合，并以 **集成** 给出不确定性。这些连接之所以干净，只有一个原因：**经典数学与 ML 是在不同分辨率上描述同一批对象。** 存活 = exp(−H)、WFR 的死亡 / 传输分解、剂量单调性、成分可识别性是 *约束*；深度编码器、FiLM 条件化、单调神经危险率、flow matching 是 *逼近器*；而非平衡传输正是两者相遇之处——几何为这个分解命名，网络则去估计它。

## 纯 ML 会在哪里跑偏

诱人的捷径是丢掉所有结构，训练一个针对处理细胞的大型条件生成器。它会奏效——而且它会重现这一领域既有的那个窟窿：你会预测 *存活者的表达*（CPA、[scGen](https://doi.org/10.1038/s41592-019-0494-8)、CellFlow 已经做得很好的事），同时悄悄丢掉死亡 / 传输的分解、剂量单调性，以及可识别性条款。这三样不是装饰；它们正是这之所以是一个 *存活* 模型、而非又一个扰动预测器的全部理由。所以纪律是定死的：**现代 ML 用于映射与条件化；经典数学用于约束与可识别性**——而非平衡传输那道接缝，正是让两者成为一个模型、而非两个模型的关键。

## 论点，一句话

老数学是骨架——约束、可识别性、外推保证。现代 ML 是肌肉——flow matching、摊销式条件化、单调神经危险率。模型就是那个 **接合**，而接合处正是 Wasserstein–Fisher–Rao 分解，在那里一个生长感知 flow 的生长 head *就是* 存活场、它的速度 head *就是* 状态转移：一个网络，条件化一次、训练一次，终于把整套现代工具箱对准它，去建模 *谁存活了下来*。
