> *耗竭随笔的第三版——v1 问的是"我们该构建什么模型"，v2 问的是"这是哪一类模型，又是谁已经解决了它"，而这一篇构建这个模型。它仍是[可达性 / 存活力纲领](cell-state-reachability-as-viability-theory.md)的、药物条件的、可经验拟合的实例，建在[扩散 / 密度骨干](diffusion-and-flow-matching-for-virtual-cells.md)之上，是[稳定性与寿命](cell-stability-and-niche-dependence-in-vc-fms.md)的 τ→0 切面，本质上是一个[因果](causal-models-fm-and-vc.md)对象，也是[智能体 + FM + 机器闭环](agent-fm-machine-closed-loop.md)天然的采集目标。优先级排序乘着[惊奇 / 不确定性信号](surprise-and-uncertainty-in-cell-fms.md)。*

## 从"作为组合而新颖"到那个组合

第二版以一个裁决收尾：生存场 π(*x*|*g*,*d*)——一个处于状态 *x* 的细胞在药物 *g*、剂量 *d* 下持存的概率——不是新数学。它是把三个成熟想法翻译进单细胞空间——可靠性工程的剂量→风险外推、生态学的占用率/仅有出现的似然，以及非平衡最优传输的死亡场——而生物学从未把它们组装到一起，并且那一个确实建模了细胞死亡的方法（Waddington-OT）无法泛化到一个未见过的药物。这是一个精确、可证伪的主张。这一版当真对待它，并把这个组合写成一个可训练的模型：这个对象是什么、损失是什么、我们导入什么对比发明什么，以及什么会证明它错了。

用户设定、本文遵循的设计规则：**通过把一个非生物学模型迁移到这个生物学问题上来构建新模型，并把最接近的单细胞方法——Waddington-OT 与 moscot——当作要击败的基线，而非要复用的引擎。** 学习它们的机械装置；不要继承它们的局限。

## 这个对象，落到实处

完全在一个冻结的嵌入空间中工作，让模型永远不必重新学习一个细胞是什么。一个细胞状态 *x* 是 [Tahoe-x1](https://www.biorxiv.org/content/10.1101/2025.10.23.683759.full.pdf) 隐向量 *z* ∈ ℝᵏ。一个药物 *g* 是其结构的冻结 [MoLFormer](https://arxiv.org/abs/2106.09553) 嵌入 *c_g*——正是这一招让 π 能为从未训练过的分子做预测。剂量 *d* 是一个标量（log-molar）。

生存场是一个 **塌缩到快照的积分风险**。每个细胞携带一个死亡风险 λ(*z*,*g*,*d*)；在固定的 24 h Tahoe 窗口 τ 上，存活分数为

> π(*z*\|*g*,*d*) = exp( −τ · λ(*z*,*g*,*d*) )。

单一快照之所以足够，恰恰是因为我们从不积分一条轨迹——我们直接从累积风险读出存活分数，正是放射生物学写下 surviving fraction = exp(−*αD*−*βD*²) 时所用的那个恒等式。

## 我们导入的引擎：可靠性工程，操作化

这是被迁移过来的非生物学模型，也是这个设计赚回成本的地方。在[加速寿命试验](https://onlinelibrary.wiley.com/doi/book/10.1002/9780470316795)中，你对一个零件施加应力，并把失效时间建模为一个被应力 *加速* 的风险，加速律以参数化形式选定，使得拟合出的风险 **外推到你从未测试过的应力水平**。把剂量映射到应力，律就自己写出来了：

> log λ(*z*,*g*,*d*) = log λ₀(*z*) + *s*(*z*, *c_g*) · log *d* + *b*(*z*, *c_g*)。

λ₀(*z*) 是一个状态依赖的基线脆弱度；*s* 与 *b* 是作用于（细胞，药物）嵌入对上的小 MLP 头部，给出剂量上的一个 **逆幂律加速**。两条性质免费掉了出来，二者都是该借用而非拟合一个黑箱的理由：风险按构造 **在剂量上单调**（没有药物在更高浓度下变得更安全），并且它 **可外推**——一个加速寿命模型的全部目的就是从低应力数据预测高应力区。放射生物学的线性-二次曲线，正是这个对象在把 *z* 与 *g* 积分掉之后的样子；我们正在还原它丢弃的那种逐状态、逐药物的分辨率。（一个 Arrhenius 式或 Cox 比例的 `exp(s(z,c_g,d))` 形式是一个即插即用的替代；逆幂律是最干净的剂量-即-应力类比。）

## 损失：非平衡 OT，其中死亡项 *即是* 生存场

我们从不观测到一次死亡。我们每孔观测到两个组成性密度：对照 p₀(*z*) 与处理后 p₁(*z*\|*g*,*d*)。生成性叙事是：处理后的样本是对照群体 **被生存稀疏化、再被传输** 后的结果：以概率 1−π 杀死每个对照细胞，然后让幸存者漂移到一个新状态。这就是、逐字地，一个耦合的[非平衡-OT](https://arxiv.org/abs/1508.05216) 分解，分成一个 *反应（增殖–死亡）* 项和一个 *传输* 项——因此训练目标是单一的非平衡-OT 散度：

> 在 θ（风险）与 T（一个传输映射）上取 min：  𝒲_ub( T#( π_θ · p₀ ) , p₁ ) + ℛ。

𝒲_ub 是一个带 KL 边际松弛的[熵正则 / Sinkhorn](https://arxiv.org/abs/1306.0895) 非平衡 OT；**边际松弛（质量销毁）分量 即是 π_θ**，而 **传输分量 即是 状态转移**，T 是一个神经 Monge 映射（[ICNN](https://arxiv.org/abs/1908.10962)，由 [Monge gap](https://arxiv.org/abs/2302.04953) 正则化），只作用于幸存者。竞争风险的读法是同一拆分换了个名字：死亡是一个因果特定风险，状态变化是另一个。我们不是在发明死亡/传输分离——moscot 已经逐条件计算它了；我们是在把它做成一个 *(g, d) 的学得函数*，而非一次逐孔拟合。

## 让它保持诚实的那一条：识别仅至一个常数

v2 把 v1 松散的告诫磨锐成一个精确的，而模型必须尊重它。因为 p₁ 是 **组成性的**——相对计数，死细胞早已离场——故每条件的总存活质量未知，所以 π 仅 **在每个 (g, d) 一个乘性常数的意义下** 被识别。这不是一个需要敷衍掩盖的麻烦；它同时是生态学的占用率常数、PU 学习的 [Elkan–Noto](https://dl.acm.org/doi/10.1145/1401890.1401920) 标注常数，以及 Heckman 选择。两种钉住它的方法，按偏好排序：(1) 一个 **绝对计数锚点**——掺入标准品（spike-ins）或逐孔细胞计数——直接固定该常数；(2) 退而求其次，一个 **占用率式联合似然**，把接近零的剂量绑定到一个生存≈1 的先验，以仅有出现的模型校准检测的方式来校准该常数。直白地说出来，好让一个审稿人能据此约束我们：**没有一个绝对锚点，只有跨状态与跨剂量的相对生存被识别**——这仍足以为脆弱状态排序、并跑通死亡/传输分解，只是不足以宣称一个绝对杀死分数。

## Waddington-OT 与 moscot：基线，不是引擎

| | Waddington-OT / moscot | 我们的 π_θ |
|---|---|---|
| 死亡/传输拆分 | 是（非平衡耦合） | 是（同一拆分，**学得的**） |
| 以药物结构为条件 | 否 | 是（MoLFormer *c_g*） |
| 预测未见药物 | **否** | 是（留出分子） |
| 预测未见剂量 | 否 | 是（ALT 外推） |
| 需要一条时间序列 | 实际上是 | 否（单一快照） |

[Waddington-OT](https://www.cell.com/cell/fulltext/S0092-8674(19)30039-X) 与非平衡 [moscot](https://www.nature.com/articles/s41586-024-08453-2) 是正确的基线，因为它们是 *唯一* 已经为每个细胞附上增殖/死亡、并把被杀死与被传输的质量拆开的单细胞方法。我们以两种方式使用它们。**分布内一致：** 逐 (*g*,*d*) 孔运行它们，并检查我们学得的 π 是否复现了它们的死亡率估计——若它在已见条件上都无法匹配专家，那么函数形式就是错的。**分布外缺口：** 它们没有协变量的函数形式，因此在一个留出的分子或一个未见的剂量上它们根本无法产出一个估计；那个缺口就是全部贡献。我们借用它们的反应/传输分解和它们的熵正则求解器，并丢弃"每个条件都要有自己拟合的耦合"这一假设。

## 在 Tahoe-100M 上训练

[Tahoe-100M](https://www.biorxiv.org/content/10.1101/2025.02.20.639398v3.full) 是那个让协变量函数形式可学习的底料：1 亿细胞、约 1,100 种药物、50 个细胞系、剂量分辨、一个 24 h 快照。冻结 Tahoe-x1（细胞编码器）与 MoLFormer（药物编码器）；只训练 λ₀、加速头部 *s*,*b*，以及传输映射 T。每个 minibatch 是一组孔；逐孔，在 π-重加权-传输后的对照密度与观测到的处理后密度之间计算熵正则-UOT 损失，并反向传播。**留出整批分子与整档剂量水平**——而非随机细胞——以便下面的泛化测试是诚实的。细胞系作为一个上下文向量进入，于是允许一个细胞系特定的基线脆弱度 λ₀。

## 如何被证伪

v1 承诺的同样四项测试，现在绑定到模型的具体部件：

- **留出孔（插值）。** 在训练网格内部的条件上，逐状态分箱预测的耗竭对观测的耗竭。
- **未见分子（承重测试）。** 仅凭 *c_g* 预测一个从未训练过的药物的耗竭模式——这是 WOT/moscot 在结构上无法做出的主张。
- **未见剂量（ALT 外推）。** 在低/中剂量上训练，预测高剂量存活分数；这里的成功是对逆幂律加速的直接检验，而非对数据的。
- **机理锚点。** 预测的逐状态死亡风险应与 BH3 谱的凋亡致敏相关——一项正交的检测，因此一致并非循环论证。

## 什么会把它弄坏

提前说出来，因为失效模式才是有趣的科学：**组成性常数**（如上）在没有锚点时不可识别；**死亡可能与非致死性的脱落混淆**（细胞周期阻滞、双联体丢失），竞争风险的死亡-对-转移拆分意在吸收它，但仅当 T 足够富表达力、能把被阻滞的细胞认领为 *移动了* 而非 *被杀死了* 时才行；**单一的 24 h 快照** 把快速杀死与缓慢杀死合并进一个积分风险，所以 τ 是一个假设、而非一次测量；以及整个事情会继承 **Tahoe-x1 的隐空间** 恰好丢弃的任何与存活相关的轴。

## 论点，一句话

v2 说生存场作为一种组合而非一个原语是新颖的。v3 是把这个组合做成一个可训练的模型：**一个可靠性工程的风险，使剂量表现得像应力——单调且可外推——其死亡-对-传输的拆分是一个非平衡-OT 耦合，跨 Tahoe-100M 拟合，以 Waddington-OT 与 moscot 作为它必须先匹配、随后在它们沉默之处击败的分布内基线。** 数学是有意借来的；新的东西是把所有这些指向那一个细胞生物学对象——谁活下来了——这个领域一直绕着它建模、而非给它建模。
