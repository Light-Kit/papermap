> *这是「消减」那篇的第三版——v1 问的是「我们该建什么模型」，v2 问的是「它属于哪一类模型、谁早就解过了」，这一篇把模型建出来。它仍然是[可达性 / 可生存性方案](cell-state-reachability-as-viability-theory.md)里那个药物条件化、可实证拟合的实例，建在[扩散 / 密度主干](diffusion-and-flow-matching-for-virtual-cells.md)之上，是[稳定性与寿命](cell-stability-and-niche-dependence-in-vc-fms.md)在 τ→0 时的那一面，本质上是个[因果](causal-models-fm-and-vc.md)对象，也是[agent + FM + 机器闭环](agent-fm-machine-closed-loop.md)天然的采集靶点。优先级由[惊讶 / 不确定性信号](surprise-and-uncertainty-in-cell-fms.md)来定。*

## 从「新在组合」到那个组合

第二版收尾时给了一个判决：生存场 π(*x*|*g*,*d*)——一个处于状态 *x* 的细胞在剂量 *d* 的药 *g* 下能续存的概率——不是新数学。它是把三套成熟想法翻译进单细胞空间——可靠性工程的剂量→危险率外推、生态学的占据率 / 仅存在似然、非平衡最优传输的死亡场——这套组合，生物学从没拼起来过；而唯一一个真去建模细胞死亡的方法（Waddington-OT），又没法推广到没见过的药。这是个精确、可证伪的论断。这一版把它当真，把那套组合写成一个可训练的模型：这对象是什么、损失是什么、哪些是我们搬来的、哪些是我们造的，以及什么能把它证伪。

用户定下、这篇也遵循的那条设计准则是：**把一个非生物学的模型迁到生物学问题上，去建新模型；而最接近的那几个单细胞方法——Waddington-OT 和 moscot——当成要超越的基线，不是拿来复用的引擎。** 学它们的机械装置，别继承它们的局限。

## 把这个对象落到实处

整件事都在一个冻结的嵌入空间里做，这样模型永远不必重新去学「细胞是什么」。一个细胞状态 *x* 就是 [Tahoe-x1](https://www.biorxiv.org/content/10.1101/2025.10.23.683759.full.pdf) 的隐向量 *z* ∈ ℝᵏ。一个药 *g* 就是它结构的冻结 [MoLFormer](https://arxiv.org/abs/2106.09553) 嵌入 *c_g*——正是这一步，让 π 能为从没训练过的分子做预测。剂量 *d* 是个标量（log 摩尔浓度）。

这个生存场是**一个塌缩到快照上的积分危险率**。每个细胞带一个死亡危险率 λ(*z*,*g*,*d*)；在固定的 24 h Tahoe 窗口 τ 内，存活分数是

> π(*z*\|*g*,*d*) = exp( −τ · λ(*z*,*g*,*d*) )。

一张快照之所以就够，恰恰是因为我们从不去积分一条轨迹——我们直接从累积危险率把存活分数读出来，这跟放射生物学写下 存活分数 = exp(−*αD*−*βD*²) 时用的是同一个恒等式。

## 我们搬来的引擎：可靠性工程，落到可操作

这就是那个被迁过来的非生物学模型，整套设计的本钱也是在这里赚回来的。在[加速寿命试验](https://onlinelibrary.wiley.com/doi/book/10.1002/9780470316795)里，你对一个零件施加应力，把失效时间建模成一个被应力*加速*的危险率，再选一条参数化的加速律，让拟合出来的危险率**能外推到你从没测过的应力水平**。把剂量映射成应力，这条律就自己写出来了：

> log λ(*z*,*g*,*d*) = log λ₀(*z*) + *s*(*z*, *c_g*) · log *d* + *b*(*z*, *c_g*)。

λ₀(*z*) 是一个依赖状态的基线脆弱度；*s* 和 *b* 是 (细胞, 药) 嵌入对上的两个小 MLP 头，给出剂量上的**逆幂律加速**。两条性质白白掉出来，而它俩正是「宁可借、不要拟合黑箱」的理由：这个危险率在构造上就**关于剂量单调**（没有药在更高浓度下变得更安全），而且它**能外推**——加速寿命模型的全部目的，就是从低应力数据去预测高应力区段。放射生物学的线性二次曲线，恰恰就是把 *z* 和 *g* 积分掉之后的这个对象；我们做的，是把它扔掉的那份逐状态、逐药物的分辨率还回来。（一个 Arrhenius 式、或者 Cox 比例式的 `exp(s(z,c_g,d))` 形式可以直接替换；逆幂律是「剂量即应力」最干净的类比。）

## 损失：非平衡 OT，而那个死亡项*就是*生存场

我们从不直接观测到一次死亡。我们每孔观测到两个成分密度：对照 p₀(*z*) 和给药 p₁(*z*\|*g*,*d*)。生成的故事是：给药样本就是对照群体**被存活削过、再被传输**的结果——以 1−π 的概率杀掉每个对照细胞，然后让幸存者漂到一个新状态。这一字不差，就是把一个耦合做[非平衡 OT](https://arxiv.org/abs/1508.05216)分解，拆成一个*反应（生长–死亡）*项和一个*传输*项——于是训练目标就是单一一个非平衡 OT 散度：

> 在 θ（危险率）和 T（一个传输映射）上取 min：  𝒲_ub( T#( π_θ · p₀ ) , p₁ ) + ℛ。

𝒲_ub 是一个带 KL 边际松弛的[熵正则 / Sinkhorn](https://arxiv.org/abs/1306.0895) 非平衡 OT；**那个边际松弛（质量销毁）分量就是 π_θ**，而**那个传输分量就是状态迁移**，T 是一个只作用在幸存者身上的神经 Monge 映射（[ICNN](https://arxiv.org/abs/1908.10962)，由 [Monge gap](https://arxiv.org/abs/2302.04953) 正则化）。竞争风险的读法，是同一道劈分换了个名字：死亡是一个起因专属危险率，状态变化是另一个。我们并不是在发明这道死亡 / 传输分离——moscot 早就逐条件地把它算出来了；我们做的，是把它变成一个*关于 (g, d) 的学习函数*，而不是逐孔的拟合。

## 让它保持诚实的那条条款：识别到一个常数为止

v2 把 v1 那条松垮的告诫磨成了一条精确的，模型必须守住它。因为 p₁ 是**成分性的**——相对计数，死掉的细胞早就不在了——每个条件下存活的总质量是未知的，所以 π 只能**识别到每个 (g, d) 一个乘性常数为止**。这不是一个糊弄过去的麻烦事；它一身三任，同时是生态学的占据率常数、PU 学习的 [Elkan–Noto](https://dl.acm.org/doi/10.1145/1401890.1401920) 标记常数、以及 Heckman 选择。钉死它有两条路，按偏好排序：（1）一个**绝对计数锚**——spike-in 或逐孔细胞计数——直接把常数定死；（2）实在没有，就用一个**占据率式的联合似然**，把接近零剂量绑到一个 存活≈1 的先验上，像仅存在模型校准检测率那样去校准这个常数。把话说白，好让审稿人拿这个卡我们：**没有绝对锚，能识别的就只是跨状态、跨剂量的相对存活**——而这仍然够用，够给脆弱状态排序、够跑那道死亡 / 传输分解，只是没法宣称一个绝对的杀灭分数。

## Waddington-OT 和 moscot：是基线，不是引擎

| | Waddington-OT / moscot | 我们的 π_θ |
|---|---|---|
| 死亡 / 传输劈分 | 有（非平衡耦合） | 有（同一道劈分，**学出来的**） |
| 以药物结构为条件 | 没有 | 有（MoLFormer *c_g*） |
| 预测没见过的药 | **不能** | 能（留出分子） |
| 预测没见过的剂量 | 不能 | 能（ALT 外推） |
| 需要时间序列 | 实质上需要 | 不需要（单张快照） |

[Waddington-OT](https://www.cell.com/cell/fulltext/S0092-8674(19)30039-X) 和非平衡的 [moscot](https://www.nature.com/articles/s41586-024-08453-2) 之所以是对的基线，是因为它们是*唯二*已经给每个细胞挂上生长 / 死亡、并把杀掉的质量和传输的质量劈开的单细胞方法。我们用它们用在两处。**分布内的吻合：** 逐 (*g*,*d*) 孔位跑一遍，看我们学到的 π 能不能复现它们的死亡率估计——要是在见过的条件上都追不平这个专家，那函数形式就是错的。**分布外的落差：** 它们没有协变量的函数形式，所以在一个留出的分子或一个没见过的剂量上，它们压根给不出估计；那道落差，就是整份贡献。我们借它们的反应 / 传输分解和它们的熵正则求解器，扔掉「每个条件都要自己拟一份耦合」那条假设。

## 在 Tahoe-100M 上训练

[Tahoe-100M](https://www.biorxiv.org/content/10.1101/2025.02.20.639398v3.full) 就是那块让协变量函数形式可学的底料：1 亿细胞、约 1,100 种药、50 个细胞系、剂量分辨、24 h 一张快照。冻结 Tahoe-x1（细胞编码器）和 MoLFormer（药物编码器）；只训 λ₀、加速头 *s*、*b*，以及传输映射 T。每个小批量是一组孔位；逐孔，在「π 重加权再传输后的对照密度」和「观测到的给药密度」之间算熵正则 UOT 损失，再反传。**整分子、整剂量档都留出来**——而不是随机留细胞——好让下面那些泛化测试是诚实的。细胞系作为一个上下文向量进来，这样允许一个细胞系专属的基线脆弱度 λ₀。

## 怎么把它证伪

还是 v1 许下的那四条测试，现在每条都钉到了具体的模型部件上：

- **留出孔位（内插）。** 在训练网格之内的条件上，按状态分箱比一比预测和观测的消减。
- **未见分子（吃重的那条）。** 只凭 *c_g*，去预测一个从没训练过的药的消减模式——这正是 WOT/moscot 在结构上做不出来的论断。
- **未见剂量（ALT 外推）。** 在低 / 中剂量上训练，预测高剂量的存活分数；这里成不成，直接检验的是逆幂律加速本身，而不是数据。
- **机理锚。** 预测的逐状态死亡危险率，应该和 BH3 profiling 的凋亡预备程度相关——这是个正交的试验，所以吻合不是循环论证。

## 什么会把它弄垮

把话说在前头，因为这些失败模式才是有意思的科学：上面说的那个**成分常数**，没有锚就识别不了；**死亡可能和非致死性脱落混淆**（细胞周期阻滞、双联体丢失），竞争风险那道「死亡 vs 迁移」劈分本是用来吸收它的，但前提是 T 足够灵活，能把那些被阻滞的细胞认成*被移动了*而不是*被杀了*；**单张 24 h 快照**把快杀和慢杀揉进了一个积分危险率里，所以 τ 是个假设，不是个测量；而整套东西，还得继承 **Tahoe-x1 的隐空间**碰巧丢掉的那些跟存活相关的轴。

## 一句话结论

v2 说，生存场新在组合，不在它是个原创对象。v3 就是那套组合做成的一个可训练模型：**一个可靠性工程的危险率，让剂量表现得像应力——单调、可外推——它那道死亡 vs 传输劈分是一个非平衡 OT 耦合，横跨 Tahoe-100M 拟合，把 Waddington-OT 和 moscot 当成分布内必须先追平、再到它们哑火的地方甩开的基线。** 这套数学是故意借来的；真正新的，是把它们整个对准那一个细胞生物学对象——谁活了下来——而这个对象，整个领域一直在绕着它建模，就是不去建模它本身。
