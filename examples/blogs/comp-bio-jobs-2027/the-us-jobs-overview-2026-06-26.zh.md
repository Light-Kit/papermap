# 美国全职总览——581 个岗，按 sponsorship 来读

全职地图讲的是**全世界**的战略；这一篇把镜头推到**美国**这半，按数字读。把 vault 筛到 `United States`，你看到的就是：**581 个全职岗**——**364 个工业界 FTE** 加 **217 个学术博后**——分在六个板块，每一个都带 Visa-sponsor 标记,因为对一个中国籍候选人来说,把这张表重新排序的是这个标记,不是科研。

## 最要紧的那道分水岭：sponsorship

总数是 **321 个 `✓` · 251 个 `?` · 9 个 `✗`**。这不是乐观——一个 `✓` 意味着有真实、有出处的依据可以替一个中国籍办身份,而它落在两堆里:

- **cap-exempt 的博后。**美国的大学和非营利研究机构是 H-1B **cap-exempt**——它们办 H-1B 是**绕开抽签**的,全年都能办,还有 J-1。这就让学术博后成了一个外国人在美国最可靠的全职落脚点,而不是退而求其次。美国一共 217 个博后,其中 199 个基于这一点标成 `✓`——MSK、Dana-Farber、Broad、Fred Hutch、Penn、那批癌症中心——很多正好对单细胞 / 空间 / HRD 的底子。
- **常规办身份的工业界。**大的、成熟的药企和大厂办 H-1B 是常态:基因泰克 gRED、诺华 Oncology Data Science、艾伯维、BMS、默克、辉瑞、安进,还有 NVIDIA / Illumina / Tempus 那一档。瞄 PhD 的**第一个台阶**——「Scientist」「Bioinformatics Scientist」「Computational Biologist I」——哪怕岗位写着「Senior」,因为 PhD+2 年往往就够线。

那 **251 个 `?`** 多半是没写政策的 techbio 初创:第一轮就问清楚 sponsorship,因为一家不肯办 H-1B 的小公司,科研再好对你也是死路。而 **9 个 `✗`** 是真焊死的门——DOE 国家实验室和联邦岗（公民/安全许可）,还有 **T32 经费的培训型博后**（按法规只收公民/绿卡）。标出来,就是免得你白投。

## 581 个怎么按板块分

- **学术 / 研究所——205 个。**cap-exempt 的发动机,大多数 `✓` 博后都在这儿。
- **ML 优先的生物技术（techbio）——181 个。**公司最多、`?` 最多;极其对口,但 sponsorship 没写明。
- **大药企 / 大型生物技术——74 个。**岗数不多,但 `✓` 最密,第一台阶的 Scientist 头衔也最干净。
- **诊断 / 测序 / 工具——89 个。**临床基因组和管线活;你的 NGS 技能能直接对上。
- **大厂——bio/health——22 个。**NVIDIA BioNeMo、微软、谷歌——名头响,从 FM 角度切真的对口。
- **国家实验室——10 个。**标了,且基本关着（公民/许可）——就是那堆 `✗`。

## 状态标签在告诉你什么

**194 个 open（抓取时有活 req）、262 个 recurring / rolling、76 个 anticipated — monitor、27 个 closed（recurs）**（另有 22 个状态未定）。那条诚实声明从地图延续过来:没有 2027–2029 的活 req,而且全职 req 烂得快,所以凡是 2026 的岗已经关了的,都被降级成「closed（recurs）」、链接改指那个组的 careers 页。没有一个死链被当成活的展示,也没有一个 URL 是编的。

## 从这儿起步

四个美国岗预先加了星当靶心:**基因泰克** gRED Computational Sciences Scientist、**诺华** Oncology Data Science、**MSK** 的 Chan 组计算癌症博后、**Fred Hutch** 的 Setty 组单细胞方法博后——一个会 sponsor 的工业界台阶,两个 cap-exempt 的地板,这正是该一起跑的搭配。筛 `United States`,再按你真正做的方向去筛,读 **full-time map** 看跨境战略,读 **how-to-read** 看每一个徽章。**international jobs overview** 是这个 vault 的另一半。
