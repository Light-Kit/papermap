# how to read the international vault

this is the non-US tracker — 193 organizations and 238 comp-bio PhD internship roles across UK & Europe, China, Singapore/Japan/Korea, and Canada/Australia. it is built for a computational-biology PhD with a Chinese (PRC) passport, currently in a US program, looking for a summer-2027 internship abroad. the US vault is separate; here is how this one works.

## the filter is region, not sector

in the US vault you filter by sector. here the **modality filter is region** — `UK & Europe` · `China` · `Singapore / Japan / Korea` · `Canada / Australia` — because outside the US the first question is whether you can legally work there, and that is decided by geography. the sector (pharma, techbio, academic, diagnostics, big-tech) is still shown, but it lives in each card's meta line rather than as the filter. the **topic** filter works the same as the US vault — `single-cell`, `spatial`, `comp-onc`, `genomics`, and so on.

## the work-authorization flag (this replaces the F-1 flag)

abroad, F-1 / CPT / OPT do not apply — those govern US work only. so every position carries a **work-auth flag for a PRC national** instead:

- **Work-auth ✓** — China roles (no work-visa barrier for a PRC national), or a non-China program that explicitly welcomes international / non-EU applicants or provides permit support (EMBL, roche RiSE, A*STAR). earned, not assumed.
- **Work-auth ?** — visa stance is unstated, or the program is open but you must secure the permit/host support yourself. this is most non-China rows. verify before you invest.
- **Work-auth ✗** — restricted to nationals / EU citizens / permanent residents, or the posting explicitly says no sponsorship (e.g. AstraZeneca UK, GSK placement). do not spend applications here.

the flag is deliberately conservative outside china, and deliberately green inside it, because for a PRC national the work-authorization reality genuinely is that different by region.

## status labels

- **open (2026 cycle)** — was live in the 2026 round when harvested.
- **closed (2026 — recurs)** — the 2026 posting closed; the program recurs; link points to the careers/program page.
- **recurring program** — a standing named scheme (RiSE, A*STAR attachment, a university summer program) rather than a single dated req.
- **anticipated 2027** — expected to run for 2027 based on history; link is the page to monitor.

## the honesty caveat

as with the US vault, there are **no live summer-2027 links here yet** — that cycle is not posted. every link points either to a 2026-cycle posting (to show the shape and eligibility of the real role) or to the organization's careers / program page (where the 2027 version will appear). a note on china: many china roles run on campus-recruitment (校招 / 生信实习) timelines and are tracked through company WeChat / 校招 portals and lab contacts rather than a western careers ATS, so the link is often the company page or program page to watch rather than a dated req.

this is a preparation asset: it tells you which regions you can actually work in, which programs are built for international students, and where the permit question is the real screen — so that when the season opens you already know where to look.
